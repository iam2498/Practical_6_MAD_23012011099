##  About the App
This Android application showcases multiple animation styles commonly used in modern mobile apps:

-  **Splash Screen**
  - Animated logo using **frame-by-frame animation**
  - Combined with **view animation** (scale + rotation)
  - Auto-navigation to MainActivity

- **Main Activity**
  - Animated **alarm clock** using AnimationDrawable
  - Pulsating **heart icon**
  - Material Design layout using CardView

---

##  Features

###  **Splash Screen Animations**
- Frame animation of the logo  
- View animations such as:
  - Scale (grow effect)
  - Rotate (spin effect)
- Animated splash runs automatically when the app launches  
- Automatically navigates to **MainActivity**

###  **Main Activity Animations**
- Animated **alarm clock** icon (frame animation)
- Pulsating **heart** animation (view animation)
- Clean UI with **Material CardView**
- Buttons for:
  - Create Alarm (UI only)
  - Cancel Alarm (UI only)
